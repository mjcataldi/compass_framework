# 03 — Career Profile: Cover Letter Generation

This file governs COMPASS career-profile cover letter artifacts.

Generate a cover letter only when the user explicitly requests one.

A cover letter must be derived from the latest approved source resume, master CV, canonical career record, Intake claim ledger when available, target job description, current COMPASS analysis when available, and user-provided constraints.

Do not invent technologies, ownership, metrics, credentials, role history, company knowledge, personal motivation, or domain experience.

Use a calm, professional, forward-looking tone.

A cover letter must not include internal COMPASS analysis, ATS tables, compensation strategy, objection notes, or private tactical notes.
